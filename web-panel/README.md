# Web Panel - Render Static Site

## Deploy

1. Render → New → Static Site
2. GitHub repo bağla
3. Ayarlar:
   - **Branch:** main
   - **Publish Directory:** web-panel

## Özellikler

- Giriş ekranı
- Dashboard (istatistikler + grafik)
- Log listesi (filtreleme + sayfalama)
- Fotoğraf galerisi
- Ayarlar (anomali türleri + email listesi)
